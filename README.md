# Smart RPP Generator Pro (DeepLearningOS)

**Smart RPP Generator Pro** adalah aplikasi berbasis web (SPA) yang dirancang khusus untuk guru di Indonesia guna menyusun Rencana Pelaksanaan Pembelajaran (RPP) secara cerdas, cepat, dan terintegrasi dengan sistem observasi PMM (Pengelolaan Kinerja Guru) terbaru.

## 🚀 Fitur Utama

- **Integrasi 8 Indikator PMM**: Otomatisasi penyisipan target perilaku spesifik (Disiplin Positif, Keteraturan Kelas, dll) ke dalam skenario pembelajaran.
- **Smart Engine Logic**: Menghasilkan tujuan pembelajaran, langkah-langkah, dan instrumen asesmen secara kontekstual hanya dengan memasukkan topik materi.
- **18+ Model Pembelajaran**: Mendukung berbagai model mulai dari PBL, PjBL, Discovery Learning, hingga Game-Based Learning.
- **Asesmen Komprehensif**: Pembuatan instrumen diagnostik, formatif (rubrik observasi), dan sumatif secara otomatis.
- **Siap Cetak (Print-Ready)**: Output dokumen terformat standar A4 yang rapi untuk kebutuhan administrasi sekolah.
- **UI/UX Modern**: Antarmuka berbasis "Glassmorphism" yang responsif dan nyaman di mata (Dark Mode).

## 🛠️ Teknologi yang Digunakan

- **React.js (v18)**: Untuk reaktivitas dan manajemen state.
- **Tailwind CSS**: Untuk styling utilitas yang modern dan responsif.
- **Lucide React**: Untuk ikonografi yang clean.
- **Babel**: Untuk kompilasi JSX langsung di browser.
- **Plus Jakarta Sans**: Font untuk tipografi profesional.

## 📂 Struktur Aplikasi

- **Tahap 1: Identifikasi**: Input data dasar, topik, dan karakteristik siswa.
- **Tahap 2: Desain**: Pemilihan indikator PMM dan model pedagogis.
- **Tahap 3: Skenario**: Penulisan detail kegiatan (Pendahuluan, Inti, Penutup).
- **Tahap 4: Asesmen & Solusi**: Perencanaan penilaian dan strategi troubleshooting kelas.
- **Tahap 5: Instrumen**: Finalisasi rubrik dan instrumen penilaian.
